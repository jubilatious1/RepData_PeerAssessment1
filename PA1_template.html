  1. Devise a strategy for filling in all of the missing values in the dataset. Create a new dataset that is equal to the original dataset but with the missing data filled in.

    Use random imputation method and code from Professor Andrew Gelman at Columbia University pg. 533 (http://www.stat.columbia.edu/~gelman/arm/missing.pdf). Create random imputation function: 

random.imp <- function (a){
missing <- is.na(a)
n.missing <- sum(missing)
a.obs <- a[!missing]
imputed <- a
imputed[missing] <- sample(a.obs, n.missing, replace=TRUE) 
return (imputed)
}
